# 🎰 Slot Master Pro (スロット攻略・分析ツール)

特定ホールの過去データを収集し、傾向分析・高設定投入パターンを可視化する自分専用の攻略ツールです。
PCでの詳細分析はもちろん、**スマホでの現場稼働**に完全対応しています。

## 🚀 主な機能

### 1. 📊 強力なデータ分析
- **日別レポート**: 毎日の差枚数、勝率、主力機種を一目で確認。
- **店長推し分析 (スマホ対応)**: 「勝率×差枚数」のバブルチャートで、設定が入りやすい機種を視覚化。
- **メーカー自動判別**: 機種名からメーカー（Sammy系、SANKYO系など）を自動でタグ付けし、メーカーごとの強さを分析。
- **不発・塊検知**: 高設定不発台や、並び（塊）での投入箇所を自動サーチ。

### 2. 📱 スマホ完全最適化
- ホール内での立ち回りを想定し、グラフや文字サイズをスマホ縦画面に最適化。
- URLを開くだけで、いつでもどこでもデータを確認可能。

### 3. 🛠 簡単カスタマイズ (`stores.txt`)
- プログラムを書き換える必要はありません。
- [cite_start]テキストファイルに行を追加するだけで、好きな店舗をいくつでも管理できます 。

### 4. 🛡 安全設計
- **収集時間ガード機能**: サーバー負荷とアクセス制限を考慮し、データ収集は **朝 8:00 〜 9:59** の間のみ実行可能です。
- 上記以外の時間は収集ボタンがロックされ、誤操作によるBANリスクを回避します。
- スクレイピング時は最新の偽装技術を使用し、安全にアクセスします。
