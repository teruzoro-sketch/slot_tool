# 🎰 Slot Master Pro

ホールデータの収集・分析・設定推測を全自動で行う、スロット専業・ガチ勢向けの最強分析ツールです。
Python + Streamlit で構築されており、ブラウザ上で直感的にデータを分析できます。

## 🔥 特徴

### 🛡️ 鉄壁のデータ収集 (Logic)
* **偽データ完全ブロック**: ホールが「ランキング表（一部の台）」しか公開していない場合、自動で検知して収集をスキップします。（網羅率80%ルール）
* **マスタ自動更新**: 過去の台番号データを蓄積し、新台入替や配置変更にも自動追従します。
* **スマートアクセス**: サーバー負荷を考慮し、指定時間帯（デフォルト: 8:00~9:59）のみ収集を許可する安全設計です。

### 📊 多彩な分析機能 (App)
1.  **日別レポート**: その日の「強末尾」「全台系」を一目で判定。
2.  **機種別詳細 (コンパクト)**: スクロール不要で大量の台データを一覧表示。ジャグラー系は **「ブドウ逆算」** と **「設定推測（虹/金/黄）」** を自動表示。
3.  **不発・塊検知**: 高設定の不発台や、並び（塊）箇所の候補を自動サーチ。
4.  **鉄板台サーチ**: 特定の条件（回転数、差枚、確率）を満たす「エース台」を抽出。
5.  **脳AI攻略**: 「凹み上げ」「据え置き」「ローテ」などの法則から、翌日の狙い台をスコアリング。

## 🛠️ インストール方法

1. リポジトリをクローン

git clone [https://github.com/your-username/slot-master-pro.git](https://github.com/your-username/slot-master-pro.git)
cd slot-master-pro

2. 依存ライブラリのインストール

pip install -r requirements.txt

3. 店舗設定ファイルの作成 stores.txt を作成し、収集したい店舗情報を記述してください。

店舗名A | [https://min-repo.com/ranking/](https://min-repo.com/ranking/)... | 7のつく日
店舗名B | [https://min-repo.com/ranking/](https://min-repo.com/ranking/)... | ゾロ目

🚀 使い方
以下のコマンドでアプリを起動します。

streamlit run app.py
ブラウザが立ち上がり、分析ダッシュボードが表示されます。 （スマホから見る場合は ngrok 等の利用を推奨）

⚙️ 設定
収集時間の変更: logic.py の SCRAPE_SAFE_START, SCRAPE_SAFE_END で変更可能。

データベース: slot_data.db にSQLite形式で保存されます。
