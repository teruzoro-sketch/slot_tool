===========================================================
 🎰 Slot Master Pro - 導入＆操作マニュアル
===========================================================

このツールは、特定店舗のスロットデータを収集・分析し、
「店長の癖」や「高設定の投入傾向」を可視化するPythonツールです。

-----------------------------------------------------------
 📁 フォルダ構成の確認
-----------------------------------------------------------
以下のファイルが同じフォルダにあることを確認してください。

[フォルダ]
  ├─ app.py              (メイン画面のプログラム)
  ├─ logic.py            (データ収集ロジック)
  ├─ requirements.txt    (必要なライブラリ一覧)
  └─ proxy_list.txt      (任意: プロキシを使う場合のみ作成)

-----------------------------------------------------------
 🚀 インストール手順 (初回のみ)
-----------------------------------------------------------
1. Pythonがインストールされていることを確認してください。
2. コマンドプロンプト(Windows) または ターミナル(Mac) を開きます。
3. このフォルダの場所まで移動します（cdコマンドなどを使用）。
4. 以下のコマンドを入力して、必要なライブラリをインストールします。

   pip install -r requirements.txt

-----------------------------------------------------------
 ▶️ 起動方法
-----------------------------------------------------------
以下のコマンドを入力すると、ブラウザが立ち上がりツールが起動します。

   streamlit run app.py

※ 初回起動時、メールアドレスの入力を求められることがありますが、
   何も入力せずに Enter を押してスキップして構いません。

-----------------------------------------------------------
 ⚠️ データ収集の注意点
-----------------------------------------------------------
・「データの更新・収集」は、サーバー負荷を考慮し【朝 8:00 〜 10:00】の間に行うことを強く推奨します。
・短時間に過度なアクセスを行うと、サイト側からブロックされる可能性があります。
・本ツールは「curl_cffi」等を使用し、ブロック回避対策を施していますが、100%の動作を保証するものではありません。
